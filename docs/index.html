inklog.js

Nodejs Logger

Install

  • yarn: yarn add inklog.js
  • npm npm i inklog.js

Usage

Name Usage Example Color Prefix
error .error() logger.error('Hello World') Red [ERROR]
info .info() logger.info('Hello World') Green [INFO]
warn .warn() logger.warn('Hello World') Yellow [WARN]
debug .debug() logger.debug('Hello World') Cyan [DEBUG]
log .log() logger.log('Hello World') None None

Development

  1. Clone Repo: gh repo clone Lucaslah/inklog.js or git clone https://github.com/lucaslah/inklog.js
  2. Install Dependencies: npm install or yarn
  3. Build Code: yarn build or npm run build
